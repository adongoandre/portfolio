// 03 — About Me

The Engineer Behind The Code

I'm a programmer with a deep passion for robotics and embedded systems. I love the intersection of hardware and software — the moment a robot moves because of code you wrote is unlike anything else.

My journey started with Arduino experiments and has grown to include full autonomous robot builds, computer vision systems, and low-level firmware development in C. I bring the same precision and attention to detail to web development with HTML and CSS.

When I'm not coding, I'm probably tinkering with circuits, reading about control theory, or building the next project. I believe in clean code, well-documented systems, and never stop learning.
